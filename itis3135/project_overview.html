Project Overview: Civilization VI Strategy Hub

Overview

This project is a multi-page educational and interactive website centered around the video game Civilization VI. Its purpose is to provide new and returning players with strategy guides, leader breakdowns, build order tips, and tools to enhance gameplay experience. This includes a dynamic tech tree viewer and a Civ selector quiz.

The intended users are Civ 6 players ranging from beginners to intermediate strategists looking to improve their gameplay or explore different playstyles.

The site includes a homepage, a leaders guide, a dynamic tech tree, a strategy resources section, and a contact/about page with community links.

Page Designs

  • index.html – Home page
    Purpose: Introduce site and feature popular content
    Audience: All users
    Content: Welcome section, featured guides
    Buttons: Scroll to guide, visit leaders page
  • leaders.html – Leader Guide
    Purpose: Provide strategy breakdowns by leader
    Audience: Intermediate players
    Content: Interactive list of leaders
    Interactivity: Filters and dropdowns
  • techtree.html – Tech Tree Explorer
    Purpose: Show tech/civic tree progression
    Audience: All players
    Content: Visual tree, interactive pop-ups
    JS Use: AJAX loading of tree data
  • strategy.html – Strategy Resources
    Purpose: Share tips, guides, videos
    Audience: All players
    Content: Embedded YouTube playlists, curated links
  • contact.html – Contact/About
    Purpose: Allow feedback + display dev info
    Audience: Users/fans
    Form: Name, email, message with validation

Dynamic Functionality

  • leaders.html: jQuery-UI dropdown to filter leaders by ability/playstyle.
  • techtree.html: AJAX loads and displays JSON-based tree layout with highlight on hover.
  • contact.html: Custom JavaScript form validation + jQuery form animations.
  • strategy.html: jQuery image carousel plugin for rotating guide images.
