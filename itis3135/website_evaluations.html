Good Website: apple.com

Appleâ€™s website is a great example of modern, professional design. It is clean, easy to use, and visually appealing.

  • Contrast: Uses high contrast between text and background for readability.
  • Repetition: Fonts, spacing, and layout are consistent across all pages.
  • Alignment: Every element is carefully aligned within a clear grid system.
  • Proximity: Related items like product names and buttons are grouped together.

Bad Website: beesbeesbees.com

This website is a chaotic meme with little to no structure. It serves as a fun joke, but is a poor example of design.

  • Contrast: Too many bright colors and fast animations make it hard to focus.
  • Repetition: There is no repetition or visual consistency.
  • Alignment: No layout or structured alignment is used.
  • Proximity: Everything is stacked together without separation.
